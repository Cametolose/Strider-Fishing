# 🎣 Strider-Fishing

---

## ⚠️ Big Fat Warning

> **Use this at your own risk.** > This is a fully automated **macro**. Hypixel absolutely bans for this. If you leave this running unattended for hours, you *will* get wiped. Don't be dumb. Only use this for educational purposes.

---

## ✨ Features

* **Auto-Gear Swap:** Automatically switches between your fishing rod and your combat axe.
* **Human-like Delays:** Uses randomized sleep intervals so it doesn't look *completely* like a robot.
* **Strider Shredder:** Triggers your axe buff, then left-clicks until the mobs are dead.

---

## ⚙️ Setup

1. Make sure you have the **Minescript** mod installed.
2. Put your fishing rod and axe in your hotbar (**Slots 1–8**).

---

## 🚀 How to Run

Drop the script into your Minescript folder write in your Minecraft chat:

```text
\fishing
